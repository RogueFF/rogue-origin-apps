# Tests

Testing utilities for the Rogue Origin Operations Hub.

## Files

- **automated-dashboard-tests.js** - Automated test suite for dashboard functionality
- **quick-visual-test.js** - Quick visual regression tests
- **package.json** - npm dependencies for testing
- **package-lock.json** - Locked dependency versions

## Running Tests

```bash
# Install dependencies
npm install

# Run automated tests
node automated-dashboard-tests.js

# Run quick visual tests
node quick-visual-test.js
```

## Test Reports

Test reports are stored in `docs/sessions/` for historical reference.
