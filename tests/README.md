# Rogue Origin Apps - Test Suite

Comprehensive Puppeteer + Jest test suite for the Rogue Origin hemp processing operations management system.

## Test Files

### New Jest Test Suites (Recommended)

1. **baseline.test.js** - Baseline tests ensuring all pages load without errors
2. **iframe-navigation.test.js** - Tests for iframe-based navigation system
3. **page-loading.test.js** - Individual page functionality tests

### Legacy Test Files

- **automated-dashboard-tests.js** - Original automated test suite
- **quick-visual-test.js** - Quick visual regression tests

## Installation

```bash
cd tests
npm install
```

## Running Tests

### Run all new Jest tests:
```bash
npm test
```

### Run specific test suite:
```bash
npm run test:baseline      # Baseline tests only
npm run test:navigation    # Navigation tests only
npm run test:pages         # Page loading tests only
```

### Run with watch mode (development):
```bash
npm run test:watch
```

### Run with coverage:
```bash
npm run test:coverage
```

### Run legacy tests:
```bash
node automated-dashboard-tests.js
node quick-visual-test.js
```
